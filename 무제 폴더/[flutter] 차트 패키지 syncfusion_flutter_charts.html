[flutter] 차트 패키지 syncfusion_flutter_charts

qkrgusqls 2023. 9. 20. 16:02

pub.dev/packages/syncfusion_flutter_charts

 


다행히 프로젝트에 사용했던 chart는 막대그래프였기 때문에, 차트를 custom할 필요가 없었다. 

 

  • project 내의 pubspec.yaml 파일에 패키지 추가해주기
  syncfusion_flutter_charts: ^22.2.9

 

  • 차트를 불러와야할 파일의 최상단에 import 해주기
import 'package:syncfusion_flutter_charts/charts.dart';

 

차트 데이터

 

class ChartData {
  final String x;
  final double y;
  final Color color;

  ChartData(this.x, this.y, this.color);
}

 

차트의 데이터를 나타내는 사용자 정의 클래스이다.
X축 레이블(x), Y축 값(y), 데이터 포인트의 색상(color)에 대한 속성이 포함되어 있다.

 

 

    setState(() {
      data = [

        ChartData(
          '점수',
          Score,
          const Color.fromARGB(255, 241, 133, 145),
        ),
      ];
    });

 

위의 클래스에 따라 차트데이터를 업데이트 해준다.

 

차트 UI

페이지 전체에 차트가 꽉차는 것을 원하지 않아서, center를 통해 중앙정렬하고 container로 감싸주었다. 

SfCartesianChart(
  primaryXAxis: CategoryAxis(
    labelStyle: const TextStyle(
  //
    ),
  ),
  primaryYAxis: NumericAxis(
    minimum: 0,
    maximum: 100,
    interval: 10,
    ),
  ),
  tooltipBehavior: _tooltip,
  series: <ChartSeries<ChartData, String>>[
    BarSeries<ChartData, String>(
      dataSource: data,
      xValueMapper: (ChartData data, _) => data.x,
      yValueMapper: (ChartData data, _) => data.y,
      name: ' ',
      pointColorMapper: (ChartData data, _) => data.color,
      dataLabelSettings: const DataLabelSettings(
        isVisible: true,
      ),
    ),
  ],
),

 

SfCartesianChart: Syncfusion Flutter Charts의 CartesianChart로 데이터를 시각적으로 표시 했다. 

primaryXAxis 및 primaryYAxis: 이 부분에서 X 축 및 Y 축을 구성한다.
X 축은 CategoryAxis로 정의되어 있으며, 레이블 스타일, 폰트 크기 및 색상을 설정할 수 있다.
Y 축은 NumericAxis로 정의되어 있으며, 최소값(minimum), 최대값(maximum), 간격(interval) 및 레이블 스타일을 설정한다.

 


tooltipBehavior: 이것은 차트에서 툴팁을 활성화하는 데 사용되는 구성 요소이다.
_tooltip 변수에 할당된 TooltipBehavior 객체를 사용한다.

툴팁을 활용하면 차트 위에 마우스를 가져다 놓았을 때, 팝업형태로 차트의 정보를 띄워줄 수 있다. 

 

 

 


xValueMapper와 yValueMapper 함수는 데이터를 차트에 매핑한다.
xValueMapper는 X 축 데이터를 매핑하고, yValueMapper는 Y 축 데이터를 매핑한다.
색상 구분을 위해 pointColorMapper는 각 데이터 포인트의 색상을 설정했다.

나중에는 custom 차트를 만들어보고 싶다!
