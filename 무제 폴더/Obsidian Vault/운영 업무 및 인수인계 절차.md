> A. 운영 업무

- 인계자: 운영 즉시 수정 건(DB 긴급 수정 등) 처리
- 인수자: 코드 분석 및 긴급 외 코드 수정 -> 인수인계 완료 -> 즉시 수정 건 처리


> B. 인수인계

- 인계자: 인수인계 문서 작성(2주 간) -> 인수인계 문서 확인(확인 주체: 김정현 이사님) -> 인수인계 진행(1일 1시간)
- 인수자: 인수인계 문서 확인 완료 -> 인수인계 진행


> 정리

1. (인계자) 운영 즉시 수정 건 처리 / (인수자) 코드 분석 및 긴급 외 코드 수정
2. (인계자) 인수인계 문서 작성
3. (인계자) 인수인계 문서 확인
4. (인계자/인수자) 인수인계 진행
5. (인수자) 운영 즉시 수정 건 처리
